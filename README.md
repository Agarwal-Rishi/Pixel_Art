# Pixel_Art

Through recreating one of the most famous art apps, I will code Python for the first time and learn basic concepts. The app will feature customizable grids, a varied color system, and clean accessible code.

## Run

```bash
pip install -r requirements.txt
python3 main.py
```

## Progress

- Customizable grids
- Varied colors (color picker buttons)
- Grid drawing
- More tools coming: pencil, eraser, filler, opacity, UI polish
